1. First thing to do

If you want to use this asset, you should download the appropriate runtime.
The runtime can be downloaded from EsotericSoftware's Github.
See the link below.

<EsotericSoftware / spine-runtimes>

2. The second thing to do

An example of a character is contained in the Prefabs folder.
You can check the animation behavior by placing this Prefabs in the scene and playing them. (Inspector> Skeleton Animation (Script))

I hope you can fix it for your game.
